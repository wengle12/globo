Guia de Publicação do Site

Parabéns pela aquisição da hospedagem!

Agora vamos publicar seu site. Siga estes passos cuidadosamente:

1 Acesse seu Painel de Controle (cPanel)

Você deve ter recebido um e-mail com os dados de acesso ao cPanel. Geralmente o acesso é através de:

  • URL: https://seusite.com.br/cpanel (substitua pelo seu domínio)
  • Ou: https://cpanel.hospedagem.com (consulte o e-mail recebido)
Exemplo de tela de login do cPanel
2 Localize o Gerenciador de Arquivos

No cPanel, procure por "Gerenciador de Arquivos" ou "File Manager".

Ícone do Gerenciador de Arquivos no cPanel
3 Acesse a Pasta Pública

No Gerenciador de Arquivos, navegue até a pasta public_html. Esta é a pasta principal onde seus arquivos devem ficar.

Importante: Se você estiver usando um domínio principal, use a pasta public_html. Se for um subdomínio, procure pela pasta específica do subdomínio.
4 Faça Upload dos Arquivos

No Gerenciador de Arquivos, clique em "Upload" e selecione o arquivo HTML que você salvou. Se você tiver múltiplos arquivos (CSS, JS, imagens), faça upload de todos.

Botão de Upload no Gerenciador de Arquivos
5 Renomeie o Arquivo Principal (se necessário)

O arquivo principal do seu site deve se chamar index.html. Se você fez upload com outro nome, renomeie-o para index.html.

6 Acesse seu Site

Abra uma nova aba no navegador e digite seu domínio (ex: https://www.wengleinformatica.com.br). Seu site deve estar funcionando!

Nota: Pode levar alguns minutos para que as alterações fiquem visíveis para todos devido ao cache e propagação de DNS.
7 Habilite o SSL/HTTPS

No cPanel, procure por "SSL/TLS" ou "Segurança". A maioria das hospedagens oferece SSL gratuito através do Let's Encrypt. Ative o SSL para seu domínio.

Seu Site Wengle Informática Pronto

Clique no botão abaixo para baixar o arquivo HTML completo do seu site. Faça upload deste arquivo para a pasta public_html no seu servidor.

Baixar Arquivo HTML

Renomeie o arquivo para index.html após o upload.

Dúvidas ou Problemas?

Se encontrar dificuldades durante o processo:

  1. Consulte a documentação ou suporte da sua hospedagem
  2. Verifique se você está usando a pasta correta (public_html)
  3. Certifique-se de que renomeou o arquivo principal para index.html
  4. Limpe o cache do seu navegador e tente acessar novamente

Próximos Passos Após a Publicação

  • Cadastre seu site no Google Search Console
  • Crie uma página para seu negócio no Google Meu Negócio
  • Configure e-mails profissionais (ex: contato@seudominio.com)
  • Monitore seu site usando o Google Analytics
